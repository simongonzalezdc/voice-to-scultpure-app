# 🚀 Quick Start: Performance Mode

## How to Use (30 seconds)

### 1. Open Performance Mode
- Click **✨ Performance Mode** button (top right header)
- Wizard drawer appears at bottom of screen

### 2. Record Each Layer

#### Step 1: Shape (Base)
```
Prompt: "Sing a long, steady tone"
Action: Sing "Ahhhh" for 5-10 seconds
Watch:  Mic meter + Hz display
```

#### Step 2: Detail (Rhythmic Ribs)
```
Prompt: "Make percussive sounds"
Action: "Ka! Ka! Ka!" or "Ts! Ts! Ts!"
Watch:  🥁 BEAT indicator pulses
Result: Ribs appear on geometry
```

#### Step 3: Glaze (Musical Colors)
```
Prompt: "Sing a melody"
Action: C-D-E-F-G or any notes
Watch:  🎵 Note name + color swatch
Result: Harmonic colors applied
```

#### Step 4: Export
```
Prompt: "Your masterpiece is ready"
Action: Download STL/GLB
```

---

## What You'll See

### Visual Feedback During Recording

**Mic Meter:** `[████████░░] 80%`  
Shows your voice input level

**Beat Indicator (Step 2):** `🥁 BEAT!`  
Flashes red when percussive sound detected

**Note Display (Step 3):** `🎵 C4 [●]`  
Shows detected note + color preview

**Layer Panel (Right Sidebar):**  
- All recorded layers listed
- Toggle visibility (👁️)
- Adjust opacity slider
- Delete layer (🗑️)

---

## 🎯 Tips for Best Results

### Shape Step (Pitch-Driven)
- ✅ Sing long, steady notes (hold 2+ seconds)
- ✅ Use comfortable vocal range (80-400 Hz)
- ❌ Don't switch notes rapidly

### Detail Step (Beat-Driven)
- ✅ Make sharp, percussive sounds
- ✅ Try: "Ka!", "Pa!", "Ts!", tongue clicks
- ❌ Don't use sustained tones

### Glaze Step (Harmonic Colors)
- ✅ Sing clear pitches (do-re-mi)
- ✅ Hold each note for 1-2 seconds
- ❌ Don't whisper or growl

---

## 🐛 Troubleshooting

**Problem:** No beat detection  
**Fix:** Make sounds louder and sharper

**Problem:** No pitch detected  
**Fix:** Sing louder and hold notes longer

**Problem:** Colors look random  
**Fix:** Sing clearer pitches in vocal range

**Problem:** Mic meter not moving  
**Fix:** Grant microphone permission in browser

## 🎨 Layer System Quick Reference

**Add Layer:** Automatically added when you stop recording  
**Undo Layer:** Click 🗑️ in LayerPanel  
**Hide Layer:** Click 👁️ to toggle visibility  
**Adjust Blend:** Drag opacity slider (0-100%)  

**Layers compose in real-time** - changes are instant!
